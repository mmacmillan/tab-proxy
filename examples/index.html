Tab Proxy Demo

This demonstrates how to synchronize or "bridge" methods between tabs. A shared dependency is required, who's method's can be bridged, which is achieved via using localStorage. Bridged methods, when fired, will trigger in all tabs with the same arguments used for each.

Open several tabs using the "open new tab" link below, and open dev tools so you can see the console output. Then click the "invoke shared event" link here; each tab's console will show the same function has been invoked in each tab, with the same args.
